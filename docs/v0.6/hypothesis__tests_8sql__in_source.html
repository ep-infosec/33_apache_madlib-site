00015 /**
00016 @addtogroup grp_stats_tests
00017 
00018 @about
00019 
00020 Hypothesis tests are used to confirm or reject a <em>“null” hypothesis</em>
00021 \f$ H_0 \f$ about the distribution of random variables, given realizations of
00022 these random variables. Since in general it is not possible to make statements
00023 with certainty, one is interested in the probability \f$ p \f$ of seeing random
00024 variates at least as extreme as the ones observed, assuming that \f$ H_0 \f$ is
00025 true. If this probability \f$ p \f$ is small, \f$ H_0 \f$ will be rejected by
00026 the test with <em>significance level</em> \f$ p \f$. Falsifying \f$ H_0 \f$ is
00027 the canonic goal when employing a hypothesis test. That is, hypothesis tests are
00028 typically used in order to substantiate that instead the <em>alternative
00029 hypothesis</em> \f$ H_1 \f$ is true.
00030 
00031 Hypothesis tests may be devided into parametric and non-parametric tests. A
00032 parametric test assumes certain distributions and makes inferences about
00033 parameters of the distributions (like, e.g., the mean of a normal distribution).
00034 Formally, there is a given domain of possible parameters \f$ \Gamma \f$ and the
00035 null hypothesis \f$ H_0 \f$ is the event that the true parameter
00036 \f$ \gamma_0 \in \Gamma_0 \f$, where \f$ \Gamma_0 \subsetneq \Gamma \f$.
00037 Non-parametric tests, on the other hand, do not assume any particular
00038 distribution of the sample (e.g., a non-parametric test may simply test if two
00039 distributions are similar).
00040 
00041 The first step of a hypothesis test is to compute a <em>test statistic</em>,
00042 which is a function of the random variates, i.e., a random variate itself.
00043 A hypothesis test relies on that the distribution of the test statistic is
00044 (approximately) known. Now, the \f$ p \f$-value is the probability of seeing a
00045 test statistic at least as extreme as the one observed, assuming that
00046 \f$ H_0 \f$ is true. In a case where the null hypothesis corresponds to a family
00047 of distributions (e.g., in a parametric test where \f$ \Gamma_0 \f$ is not a
00048 singleton set), the \f$ p \f$-value is the supremum, over all possible
00049 distributions according to the null hypothesis, of these probabilities.
00050 
00051 @input
00052 
00053 Input data is assumed to be normalized with all values stored row-wise. In
00054 general, the following inputs are expected.
00055 
00056 One-sample tests expect the following form:
00057 <pre>{TABLE|VIEW} <em>source</em> (
00058     ...
00059     <em>value</em> DOUBLE PRECISION
00060     ...
00061 )</pre>
00062 
00063 Two-sample tests expect the following form:
00064 <pre>{TABLE|VIEW} <em>source</em> (
00065     ...
00066     <em>first</em> BOOLEAN,
00067     <em>value</em> DOUBLE PRECISION
00068     ...
00069 )</pre>
00070 Here, \c first indicates whether a value is from the first (if \c TRUE) or the
00071 second sample (if \c FALSE).
00072 
00073 Many-sample tests expect the following form:
00074 <pre>{TABLE|VIEW} <em>source</em> (
00075     ...
00076     <em>group</em> INTEGER,
00077     <em>value</em> DOUBLE PRECISION
00078     ...
00079 )</pre>
00080 
00081 @usage
00082 
00083 All tests are implemented as aggregate functions. The non-parametric
00084 (rank-based) tests are implemented as ordered aggregate functions and thus
00085 necessitate an <tt>ORDER BY</tt> clause. In the following, the most simple
00086 forms of usage are given. Specific function signatures, as described in
00087 \ref hypothesis_tests.sql_in, may ask for more arguments or for a different
00088 <tt>ORDER BY</tt> clause.
00089 
00090 - Run a parametric one-sample test:
00091   <pre>SELECT <em>test</em>(<em>value</em>) FROM <em>source</em></pre>
00092 - Run a parametric two-sample test:
00093   <pre>SELECT <em>test</em>(<em>first</em>, <em>value</em>) FROM <em>source</em></pre>
00094 - Run a non-parametric one-sample test:
00095   <pre>SELECT <em>test</em>(<em>value</em> ORDER BY <em>value</em>) FROM <em>source</em></pre>
00096 - Run a non-parametric two-sample test:
00097   <pre>SELECT <em>test</em>(<em>first</em>, <em>value</em> ORDER BY <em>value</em>) FROM <em>source</em></pre>
00098 
00099 @examp
00100 
00101 See \ref hypothesis_tests.sql_in for examples for each of the aggregate
00102 functions.
00103 
00104 @literature
00105 
00106 [1] M. Hollander, D. Wolfe: <em>Nonparametric Statistical Methods</em>,
00107     2nd edition, Wiley, 1999
00108 
00109 [2] E. Lehmann, J. Romano: <em>Testing Statistical Hypotheses</em>, 3rd edition,
00110     Springer, 2005
00111 
00112 @sa File hypothesis_tests.sql_in documenting the SQL functions.
00113 */

00160 /**
00161  * @brief Perform one-sample or dependent paired Student t-test
00162  *
00163  * Given realizations \f$ x_1, \dots, x_n \f$ of i.i.d. random variables
00164  * \f$ X_1, \dots, X_n \sim N(\mu, \sigma^2) \f$ with unknown parameters \f$ \mu \f$ and
00165  * \f$ \sigma^2 \f$, test the null hypotheses \f$ H_0 : \mu \leq 0 \f$ and
00166  * \f$ H_0 : \mu = 0 \f$.
00167  *
00168  * @param value Value of random variate \f$ x_i \f$
00169  *
00170  * @return A composite value as follows. We denote by \f$ \bar x \f$ the
00171  *     \ref sample_mean "sample mean" and by \f$ s^2 \f$ the
00172  *     \ref sample_variance "sample variance".
00173  *  - <tt>statistic FLOAT8</tt> - Statistic
00174  *    \f[
00175  *        t = \frac{\sqrt n \cdot \bar x}{s}
00176  *    \f]
00177  *    The corresponding random
00178  *    variable is Student-t distributed with
00179  *    \f$ (n - 1) \f$ degrees of freedom.
00180  *  - <tt>df FLOAT8</tt> - Degrees of freedom \f$ (n - 1) \f$
00181  *  - <tt>p_value_one_sided FLOAT8</tt> - Lower bound on one-sided p-value.
00182  *    In detail, the result is \f$ \Pr[\bar X \geq \bar x \mid \mu = 0] \f$,
00183  *    which is a lower bound on
00184  *    \f$ \Pr[\bar X \geq \bar x \mid \mu \leq 0] \f$. Computed as
00185  *    <tt>(1.0 - \ref students_t_cdf "students_t_cdf"(statistic))</tt>.
00186  *  - <tt>p_value_two_sided FLOAT8</tt> - Two-sided p-value, i.e.,
00187  *    \f$ \Pr[ |\bar X| \geq |\bar x| \mid \mu = 0] \f$. Computed as
00188  *    <tt>(2 * \ref students_t_cdf "students_t_cdf"(-abs(statistic)))</tt>.
00189  *
00190  * @usage
00191  *  - One-sample t-test: Test null hypothesis that the mean of a sample is at
00192  *    most (or equal to, respectively) \f$ \mu_0 \f$:
00193  *    <pre>SELECT (t_test_one(<em>value</em> - <em>mu_0</em>)).* FROM <em>source</em></pre>
00194  *  - Dependent paired t-test: Test null hypothesis that the mean difference
00195  *    between the first and second value in each pair is at most (or equal to,
00196  *    respectively) \f$ \mu_0 \f$:
00197  *    <pre>SELECT (t_test_one(<em>first</em> - <em>second</em> - <em>mu_0</em>)).* FROM <em>source</em></pre>
00198  */

00200     /*+ value */ DOUBLE PRECISION) (

00226 /**
00227  * @brief Perform two-sample pooled (i.e., equal variances) Student t-test
00228  *
00229  * Given realizations \f$ x_1, \dots, x_n \f$ and \f$ y_1, \dots, y_m \f$ of
00230  * i.i.d. random variables \f$ X_1, \dots, X_n \sim N(\mu_X, \sigma^2) \f$ and
00231  * \f$ Y_1, \dots, Y_m \sim N(\mu_Y, \sigma^2) \f$ with unknown parameters
00232  * \f$ \mu_X, \mu_Y, \f$ and \f$ \sigma^2 \f$, test the null hypotheses
00233  * \f$ H_0 : \mu_X \leq \mu_Y \f$ and \f$ H_0 : \mu_X = \mu_Y \f$.
00234  *
00235  * @param first Indicator whether \c value is from first sample
00236  *     \f$ x_1, \dots, x_n \f$ (if \c TRUE) or from second sample
00237  *     \f$ y_1, \dots, y_m \f$ (if \c FALSE)
00238  * @param value Value of random variate \f$ x_i \f$ or \f$ y_i \f$
00239  *
00240  * @return A composite value as follows. We denote by \f$ \bar x, \bar y \f$
00241  *     the \ref sample_mean "sample means" and by \f$ s_X^2, s_Y^2 \f$ the
00242  *     \ref sample_variance "sample variances".
00243  *  - <tt>statistic FLOAT8</tt> - Statistic
00244  *    \f[
00245  *        t = \frac{\bar x - \bar y}{s_p \sqrt{1/n + 1/m}}
00246  *    \f]
00247  *    where
00248  *    \f[
00249  *        s_p^2 = \frac{\sum_{i=1}^n (x_i - \bar x)^2
00250  *                         + \sum_{i=1}^m (y_i - \bar y)^2}
00251  *                     {n + m - 2}
00252  *    \f]
00253  *    is the <em>pooled variance</em>.
00254  *    The corresponding random
00255  *    variable is Student-t distributed with
00256  *    \f$ (n + m - 2) \f$ degrees of freedom.
00257  *  - <tt>df FLOAT8</tt> - Degrees of freedom \f$ (n + m - 2) \f$
00258  *  - <tt>p_value_one_sided FLOAT8</tt> - Lower bound on one-sided p-value.
00259  *    In detail, the result is \f$ \Pr[\bar X - \bar Y \geq \bar x - \bar y \mid \mu_X = \mu_Y] \f$,
00260  *    which is a lower bound on
00261  *    \f$ \Pr[\bar X - \bar Y \geq \bar x - \bar y \mid \mu_X \leq \mu_Y] \f$.
00262  *    Computed as
00263  *    <tt>(1.0 - \ref students_t_cdf "students_t_cdf"(statistic))</tt>.
00264  *  - <tt>p_value_two_sided FLOAT8</tt> - Two-sided p-value, i.e.,
00265  *    \f$ \Pr[ |\bar X - \bar Y| \geq |\bar x - \bar y| \mid \mu_X = \mu_Y] \f$.
00266  *    Computed as
00267  *    <tt>(2 * \ref students_t_cdf "students_t_cdf"(-abs(statistic)))</tt>.
00268  *
00269  * @usage
00270  *  - Two-sample pooled t-test: Test null hypothesis that the mean of the first
00271  *    sample is at most (or equal to, respectively) the mean of the second
00272  *    sample:
00273  *    <pre>SELECT (t_test_pooled(<em>first</em>, <em>value</em>)).* FROM <em>source</em></pre>
00274  */
00275 CREATE AGGREGATE MADLIB_SCHEMA.t_test_two_pooled(
00276     /*+ "first" */ BOOLEAN,
00277     /*+ "value" */ DOUBLE PRECISION) (
00278 
00279     SFUNC=MADLIB_SCHEMA.t_test_two_transition,
00280     STYPE=DOUBLE PRECISION[],
00281     FINALFUNC=MADLIB_SCHEMA.t_test_two_pooled_final,
00282     m4_ifdef(<!__GREENPLUM__!>,<!PREFUNC=MADLIB_SCHEMA.t_test_merge_states,!>)
00283     INITCOND='{0,0,0,0,0,0,0}'

00293 /**
00294  * @brief Perform unpooled (i.e., unequal variances) t-test (also known as
00295  *     Welch's t-test)
00296  *
00297  * Given realizations \f$ x_1, \dots, x_n \f$ and \f$ y_1, \dots, y_m \f$ of
00298  * i.i.d. random variables \f$ X_1, \dots, X_n \sim N(\mu_X, \sigma_X^2) \f$ and
00299  * \f$ Y_1, \dots, Y_m \sim N(\mu_Y, \sigma_Y^2) \f$ with unknown parameters
00300  * \f$ \mu_X, \mu_Y, \sigma_X^2, \f$ and \f$ \sigma_Y^2 \f$, test the null
00301  * hypotheses \f$ H_0 : \mu_X \leq \mu_Y \f$ and \f$ H_0 : \mu_X = \mu_Y \f$.
00302  *
00303  * @param first Indicator whether \c value is from first sample
00304  *     \f$ x_1, \dots, x_n \f$ (if \c TRUE) or from second sample
00305  *     \f$ y_1, \dots, y_m \f$ (if \c FALSE)
00306  * @param value Value of random variate \f$ x_i \f$ or \f$ y_i \f$
00307  *
00308  * @return A composite value as follows. We denote by \f$ \bar x, \bar y \f$
00309  *     the \ref sample_mean "sample means" and by \f$ s_X^2, s_Y^2 \f$ the
00310  *     \ref sample_variance "sample variances".
00311  *  - <tt>statistic FLOAT8</tt> - Statistic
00312  *    \f[
00313  *        t = \frac{\bar x - \bar y}{\sqrt{s_X^2/n + s_Y^2/m}}
00314  *    \f]
00315  *    The corresponding random variable is approximately Student-t distributed
00316  *    with
00317  *    \f[
00318  *        \frac{(s_X^2 / n + s_Y^2 / m)^2}{(s_X^2 / n)^2/(n-1) + (s_Y^2 / m)^2/(m-1)}
00319  *    \f]
00320  *    degrees of freedom (Welch–Satterthwaite formula).
00321  *  - <tt>df FLOAT8</tt> - Degrees of freedom (as above)
00322  *  - <tt>p_value_one_sided FLOAT8</tt> - Lower bound on one-sided p-value.
00323  *    In detail, the result is \f$ \Pr[\bar X - \bar Y \geq \bar x - \bar y \mid \mu_X = \mu_Y] \f$,
00324  *    which is a lower bound on
00325  *    \f$ \Pr[\bar X - \bar Y \geq \bar x - \bar y \mid \mu_X \leq \mu_Y] \f$.
00326  *    Computed as
00327  *    <tt>(1.0 - \ref students_t_cdf "students_t_cdf"(statistic))</tt>.
00328  *  - <tt>p_value_two_sided FLOAT8</tt> - Two-sided p-value, i.e.,
00329  *    \f$ \Pr[ |\bar X - \bar Y| \geq |\bar x - \bar y| \mid \mu_X = \mu_Y] \f$.
00330  *    Computed as
00331  *    <tt>(2 * \ref students_t_cdf "students_t_cdf"(-abs(statistic)))</tt>.
00332  *
00333  * @usage
00334  *  - Two-sample unpooled t-test: Test null hypothesis that the mean of the
00335  *    first sample is at most (or equal to, respectively) the mean of the second
00336  *    sample:
00337  *    <pre>SELECT (t_test_unpooled(<em>first</em>, <em>value</em>)).* FROM <em>source</em></pre>
00338  */

00467  * hypothesis \f$ H_0 : p = p^0 \f$.
00468  *

00768 /**
00769  * @brief Perform Wilcoxon-Signed-Rank test
00770  *
00771  * Given realizations \f$ x_1, \dots, x_n \f$ of i.i.d. random variables
00772  * \f$ X_1, \dots, X_n \f$ with unknown mean \f$ \mu \f$, test the null
00773  * hypotheses \f$ H_0 : \mu \leq 0 \f$ and \f$ H_0 : \mu = 0 \f$.
00774  *
00775  * @param value Value of random variate \f$ x_i \f$ or \f$ y_i \f$. Values of 0
00776  *     are ignored (i.e., they do not count towards \f$ n \f$).
00777  * @param precision The precision \f$ \epsilon_i \f$ with which value is known.
00778  *     The precision determines the handling of ties. The current value
00779  *     \f$ v_i \f$ is regarded a tie with the previous value \f$ v_{i-1} \f$ if
00780  *     \f$ v_i - \epsilon_i \leq \max_{j=1, \dots, i-1} v_j + \epsilon_j \f$.
00781  *     If \c precision is negative, then it will be treated as
00782  *     <tt>value * 2^(-52)</tt>. (Note that \f$ 2^{-52} \f$ is the machine
00783  *     epsilon for type <tt>DOUBLE PRECISION</tt>.)
00784  *
00785  * @return A composite value:
00786  *  - <tt>statistic FLOAT8</tt> - statistic computed as follows. Let
00787  *    \f$
00788  *        w^+ = \sum_{i \mid x_i > 0} r_i
00789  *    \f$
00790  *    and
00791  *    \f$
00792  *        w^- = \sum_{i \mid x_i < 0} r_i
00793  *    \f$
00794  *    be the <em>signed rank sums</em> where
00795  *    \f[
00796  *        r_i
00797  *        =   \{ j \mid |x_j| < |x_i| \}
00798  *        +   \frac{\{ j \mid |x_j| = |x_i| \} + 1}{2}.
00799  *    \f]
00800  *    The Wilcoxon signed-rank statistic is \f$ w = \min \{ w^+, w^- \} \f$.
00801  *  - <tt>rank_sum_pos FLOAT8</tt> - rank sum of all positive values, i.e., \f$ w^+ \f$
00802  *  - <tt>rank_sum_neg FLOAT8</tt> - rank sum of all negative values, i.e., \f$ w^- \f$
00803  *  - <tt>num BIGINT</tt> - number \f$ n \f$ of non-zero values
00804  *  - <tt>z_statistic FLOAT8</tt> - z-statistic
00805  *    \f[
00806  *       z = \frac{w^+ - \frac{n(n+1)}{4}}
00807  *               {\sqrt{\frac{n(n+1)(2n+1)}{24}
00808  *                - \sum_{i=1}^n \frac{t_i^2 - 1}{48}}}
00809  *    \f]
00810  *    where \f$ t_i \f$ is the number of
00811  *    values with absolute value equal to \f$ |x_i| \f$. The corresponding
00812  *    random variable is approximately standard normally distributed.
00813  *  - <tt>p_value_one_sided FLOAT8</tt> - One-sided p-value i.e.,
00814  *    \f$ \Pr[Z \geq z \mid \mu \leq 0] \f$. Computed as
00815  *    <tt>(1.0 - \ref normal_cdf "normal_cdf"(z_statistic))</tt>.
00816  *  - <tt>p_value_two_sided FLOAT8</tt> - Two-sided p-value, i.e.,
00817  *    \f$ \Pr[ |Z| \geq |z| \mid \mu = 0] \f$. Computed as
00818  *    <tt>(2 * \ref normal_cdf "normal_cdf"(-abs(z_statistic)))</tt>.
00819  *
00820  * @usage
00821  *  - One-sample test: Test null hypothesis that the mean of a sample is at
00822  *    most (or equal to, respectively) \f$ \mu_0 \f$:
00823  *    <pre>SELECT (wsr_test(<em>value</em> - <em>mu_0</em> ORDER BY abs(<em>value</em>))).* FROM <em>source</em></pre>
00824  *  - Dependent paired test: Test null hypothesis that the mean difference
00825  *    between the first and second value in a pair is at most (or equal to,
00826  *    respectively) \f$ \mu_0 \f$:
00827  *    <pre>SELECT (wsr_test(<em>first</em> - <em>second</em> - <em>mu_0</em> ORDER BY abs(<em>first</em> - <em>second</em>))).* FROM <em>source</em></pre>
00828  *    If correctly determining ties is important (e.g., you may want to do so
00829  *    when comparing to software products that take \c first, \c second,
00830  *    and \c mu_0 as individual parameters), supply the precision parameter.
00831  *    This can be done as follows:
00832  *    <pre>SELECT (wsr_test(
00833     <em>first</em> - <em>second</em> - <em>mu_0</em>,
00834     3 * 2^(-52) * greatest(first, second, mu_0)
00835     ORDER BY abs(<em>first</em> - <em>second</em>)
00836 )).* FROM <em>source</em></pre>
00837  *    Here \f$ 2^{-52} \f$ is the machine epsilon, which we scale to the
00838  *    magnitude of the input data and multiply with 3 because we have a sum with

00908 /**
00909  * @brief Perform one-way analysis of variance
00910  *
00911  * Given realizations
00912  * \f$ x_{1,1}, \dots, x_{1, n_1}, x_{2,1}, \dots, x_{2,n_2}, \dots, x_{k,n_k} \f$
00913  * of i.i.d. random variables \f$ X_{i,j} \sim N(\mu_i, \sigma^2) \f$ with
00914  * unknown parameters \f$ \mu_1, \dots, \mu_k \f$ and \f$ \sigma^2 \f$, test the
00915  * null hypotheses \f$ H_0 : \mu_1 = \dots = \mu_k \f$.
00916  *
00917  * @param group Group which \c value is from. Note that \c group can assume
00918  *     arbitary value not limited to a continguous range of integers.
00919  * @param value Value of random variate \f$ x_{i,j} \f$
00920  *
00921  * @return A composite value as follows. Let \f$ n := \sum_{i=1}^k n_i \f$ be
00922  *     the total size of all samples. Denote by \f$ \bar x \f$ the grand
00923  *     \ref sample_mean "mean", by \f$ \overline{x_i} \f$ the group
00924  *     \ref sample_mean "sample means", and by \f$ s_i^2 \f$ the group
00925  *     \ref sample_variance "sample variances".
00926  *  - <tt>sum_squares_between DOUBLE PRECISION</tt> - sum of squares between the
00927  *    group means, i.e.,
00928  *    \f$
00929  *        \mathit{SS}_b = \sum_{i=1}^k n_i (\overline{x_i} - \bar x)^2.
00930  *    \f$
00931  *  - <tt>sum_squares_within DOUBLE PRECISION</tt> - sum of squares within the
00932  *    groups, i.e.,
00933  *    \f$
00934  *        \mathit{SS}_w = \sum_{i=1}^k (n_i - 1) s_i^2.
00935  *    \f$
00936  *  - <tt>df_between BIGINT</tt> - degree of freedom for between-group variation \f$ (k-1) \f$
00937  *  - <tt>df_within BIGINT</tt> - degree of freedom for within-group variation \f$ (n-k) \f$
00938  *  - <tt>mean_squares_between DOUBLE PRECISION</tt> - mean square between
00939  *    groups, i.e.,
00940  *    \f$
00941  *        s_b^2 := \frac{\mathit{SS}_b}{k-1}
00942  *    \f$
00943  *  - <tt>mean_squares_within DOUBLE PRECISION</tt> - mean square within
00944  *    groups, i.e.,
00945  *    \f$
00946  *        s_w^2 := \frac{\mathit{SS}_w}{n-k}
00947  *    \f$
00948  *  - <tt>statistic DOUBLE PRECISION</tt> - Statistic computed as
00949  *    \f[
00950  *        f = \frac{s_b^2}{s_w^2}.
00951  *    \f]
00952  *    This statistic is Fisher F-distributed with \f$ (k-1) \f$ degrees of
00953  *    freedom in the numerator and \f$ (n-k) \f$ degrees of freedom in the
00954  *    denominator.
00955  *  - <tt>p_value DOUBLE PRECISION</tt> - p-value, i.e.,
00956  *    \f$ \Pr[ F \geq f \mid H_0] \f$.
00957  *
00958  * @usage
00959  *  - Test null hypothesis that the mean of the all samples is equal:
00960  *    <pre>SELECT (one_way_anova(<em>group</em>, <em>value</em>)).* FROM <em>source</em></pre>
00961  */
00962 CREATE AGGREGATE MADLIB_SCHEMA.one_way_anova(
00963     /*+ group */ INTEGER,
00964     /*+ value */ DOUBLE PRECISION) (
00965 
00966     SFUNC=MADLIB_SCHEMA.one_way_anova_transition,
00967     STYPE=DOUBLE PRECISION[],
00968     FINALFUNC=MADLIB_SCHEMA.one_way_anova_final,
00969     m4_ifdef(<!__GREENPLUM__!>,<!PREFUNC=MADLIB_SCHEMA.one_way_anova_merge_states,!>)
00970     INITCOND='{0,0}'
00971 );
